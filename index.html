How to use this calculator (click to expand)
  1. Choose a fundraiser type: Apex-managed, DIY event, or Typical PTO.
  2. Enter students, participation, and average donation.
  3. Optional: enter a goal to see the donation needed per participating student.
  4. Click Compare Other Fundraisers for directional comparisons.
Pro tip: Apex-managed events typically raise much more than DIY, so starting with the right baseline matters.
Assumptions used for comparisons (click to expand)
The comparison table uses your estimate as an event baseline and applies typical gross differences by fundraiser type. Program support can influence outcomes, so “on-campus team + incentives” applies a modest uplift to event gross in comparisons.
What Apex-style programs usually include
  • Event wearables / shirts
  • Leadership lessons / classroom materials
  • On-campus athletes/coaches
  • Prizes and incentives
  • Event-day production + logistics
Comparison gross multipliers (anchored to Apex baseline)
  • Apex-managed event: 1.00×
  • DIY event: 0.33×
  • Read-a-thon: 0.45× of Apex-managed
  • Product Sale: 0.35× of Apex-managed
  • Restaurant Night: separate small estimate
If “on-campus team + incentives” is set to Yes, the event baseline is lifted by +30% for comparison purposes.
