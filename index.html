How to use this calculator (click to expand)
  1. Pick a Fundraiser Style for a realistic starting point.
  2. Enter your student count.
  3. Estimate your participation rate.
  4. Enter an average donation per participating student.
  5. Optional: enter a fundraising goal.
  6. Click Compare Other Fundraisers for directional comparisons.
Pro tip: If results feel too high/low, tweak participation and average donation to match your community.
Assumptions used for comparisons (click to expand)
The comparison table uses your estimate as a Fun Run baseline and applies typical gross differences by fundraiser type. These are planning assumptions you can tune later.
Gross multipliers (relative to Fun Run)
  • Fun Run: 1.00× (baseline)
  • Read-a-thon: 0.45×
  • Product Sale: 0.35×
  • Restaurant Night: separate estimate
Fun Run (Managed Program)
Takes Fun Run gross, then subtracts:
  • 3.5% payment/processing estimate
  • 48% program split estimate (school keeps ~52%)
Read-a-thon
Takes 0.45× of Fun Run gross, then subtracts:
  • 3.5% processing estimate
  • 10% platform/program fee estimate
  • $100 fixed supplies/admin
  • $0.50 per participating student
Product Sale
Takes 0.35× of Fun Run gross, then subtracts:
  • 2% processing estimate
  • 55% product cost/margin estimate
Restaurant Night
Separate estimate:
  • 25% of participating families attend
  • $18 average spend
  • 20% giveback rate
  • $50 fixed promo/materials
Tip: Use comparisons directionally—not as guarantees.
